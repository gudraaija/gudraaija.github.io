Studija kursa anotācija

Kursa mērķis ir iegūt iemaņas SQL valodas praktiskā lietošanā. Kurss ietver sevī gan teorētisko, gan praktisko daļu. Katra teorētiskā tēma tiek nostiprināta ar atbilstošu praktisko darbu. Piemēru demonstrēšanai un uzdevumu risināšanai tiek piedāvāta datu bāzu pārvaldības sistēma Microsoft SQL Server 2012 un Transact SQL sintakse.

Studiju rezultāti

Studenti pārzin MS SQL Server jēdzienus. (eb11, eb12)
Studentiem ir apgūtas zināšanas par SQL Servera arhitektūru un funkcionalitāti. Studenti spēj praktiski strādāt ar MS SQL Server Management Studio rīku, izmantot gan rīka grafiskas saskarnes iespējas, gan arī rakstīt komandas valodā Transact-SQL. (eb31, eb33, eb35)
Studenti prot lietot MS SQL Server datu bāzu izveidei un pārvaldībai. Studentiem ir priekšstats par datu glabāšanas organizāciju SQL Serverī. (eb31, eb33, eb35)
Studenti spēj praktiski veidot un konfigurēt datubāzes objektus, saprot dažādu veidu indeksu lietošanas mērķus un integritātes ierobežojumu darbības principus, prot realizēt biznesa funkcionalitāti trigeros, glabātajās procedūrās un funkcijās, spēj pārvaldīt datubāzes transakcijas un veikt rezerves kopēšanu un atjaunošanu. (eb31, eb35, eb55)

Prasības kredītpunktu iegūšanai

Kursa laikā studentiem ir:

  1. jāizpilda praktiskie darbi par katru kursa tēmu (50% no atzīmes)
  2. jānokārto rakstisks eksāmens (30% no atzīmes)
  3. jāizpilda patstāvīgie uzdevumi (20% no atzīmes)

Lai iegūtu kredītpunktus par kursu, studentiem jānokārto eksāmens ar sekmīgo atzīmi un jāieskaita visi praktiskie darbi. Studentiem arī jāaizpilda LUIS anketa ar kursa novērtējumu (0% no atzīmes).

Kursa saturs

Tēma Nr. 1. Datu bāzu un datu bāzu failu izveide.
Datu bāzu izveide. Failu grupu izveide. Shēmu izveide. Datu bāzu snapshots izveide.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 2. Datu tipu un tabulu izveide.
Datu tipu izveide. Tabulu izveide. Partīcijās dalītu tabulu izveide.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 3. Datu integritātes nodrošināšana izmantojot ierobežojumus.
Ievads datu integritātē. Ierobežojumu (Constriants) veidi un izveide.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 4. Indeksu izveide un optimizēšana.
Indeksu plānošana. Indeksu izveide. Indeksu optimizēšana. XML indeksu izveide.
Lekcija (2 stundas). Praktiskaie darbi (2 stundas).

Tēma Nr. 5. XML izmantošana.
Datu atlase izmantojot FOR XML. Datu atlase izmantojot OPENXML. XQuery izmantošana. XML datu tipa izmantošana. XML shēmu izveide.
Lekcija (2 stundas). Praktiskaie darbi (2 stundas).

Tēma Nr. 6. Skatu izveide.
Ievads skatos. Skatu iaveide un pārvaldība. Ātrdarbības optimizācija lietojot skatus.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 7. Glabāto procedūru izveide.
Glabāto procedūru izveide. Glabāto procedūru ar parametriem izveide. Darbs ar izpildes plāniem. Kļūdu apstrāde.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 8. Funkciju izveide.
Funkciju izveide un lietošana. Darbs ar funkcijām. Funkciju izpildes konteksta noteikšana.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).
Tēma Nr. 9. Trigeru izmantošana reaģējot uz datu manipulācijas operācijām.
Trigeru izveide un izmantošana.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 10. .NET komponenšu izmantošana SQL Server datu bāzē.
Ievads SQL Servera Common Language Runtime. Asembliju imports un konfigurēšana un to sastāvdaļām atbilstošu datu bāzes objektu izveide.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 11. Transakciju un bloķēšanas pārvaldība.
Ievads transakcijās un bloķēšanā. Transakciju pārvaldība. Ievads SQL Servera bloķēšanas mehānismā. Bloķēšanas pārvaldība.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 12. Rezerves kopēšanu izveide un atjaunošana no tām.
Rezerves kopēšanas veidi, stratēģijas. Rezerves kopiju izveide. Datu bāzes atjaunošana no citas datu bāzes.
Lekcija (1 stunda). Praktiskaie darbi (1 stunda).

Tēma Nr. 13. Pamatatmiņas datu bāzes iespējas.
Bufera pūla paplašinājums. Columnstore indeksi. Atmiņas-optimizētas tabulas un darbs ar tām.
Lekcija (2 stundas). Praktiskaie darbi (2 stundas).
