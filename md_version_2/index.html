Datu bāzu praktikums

Kursa mērķis ir iegūt iemaņas SQL valodas praktiskā lietošanā. Kurss ietver sevī gan teorētisko, gan praktisko daļu. Katra teorētiskā tēma tiek nostiprināta ar atbilstošu praktisko darbu. Piemēru demonstrēšanai un uzdevumu risināšanai tiek piedāvāta datu bāzu pārvaldības sistēma Microsoft SQL Server 2012 un Transact SQL sintakse.

Prasības

Kursa laikā studentiem ir:

  1. jāizpilda praktiskie darbi par katru kursa tēmu (50% no atzīmes)
  2. jānokārto rakstisks eksāmens (30% no atzīmes)
  3. jāizpilda patstāvīgie uzdevumi (20% no atzīmes)

Lai iegūtu kredītpunktus par kursu, studentiem jānokārto eksāmens ar sekmīgu atzīmi un jāieskaita visi praktiskie darbi. Studentiem arī jāaizpilda LUIS anketa ar kursa novērtējumu (0% no atzīmes).

Studiju rezultāti

Studenti pārzin MS SQL Server jēdzienus. Kā arī ir apgūtas zināšanas par SQL Servera arhitektūru un funkcionalitāti. Spēja praktiski strādāt ar MS SQL Server Management Studio rīku, izmantot gan rīka grafiskas saskarnes iespējas, gan arī rakstīt komandas valodā Transact-SQL.
Studenti saprot dažādu veidu indeksu lietošanas mērķus un integritātes ierobežojumu darbības principus, prot realizēt biznesa funkcionalitāti trigeros, glabātajās procedūrās un funkcijās, spēj pārvaldīt datubāzes transakcijas un veikt rezerves kopēšanu un atjaunošanu.
